# Slide Road

Slide Road es una aplicación creada en Flutter que simula un juego tipo "slider" donde un auto se desplaza entre carriles a través de distintos escenarios.

---

## Tecnologías Utilizadas
- Flutter SDK  
- Flame Game Engine  
- Supabase  
- Visual Studio Code
